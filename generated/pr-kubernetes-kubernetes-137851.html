Dependency Update: Switch to google.golang.org/protobuf and etcd Vendor Changes

This PR updates multiple Go module dependencies, notably switching etcd-related dependencies to a custom fork and updating to use google.golang.org/protobuf. It also bumps several other dependencies and updates license files accordingly.

Key Changes

  • Replaced go.etcd.io/etcd dependencies with github.com/fuweid/etcd fork in code-generator examples.
  • Updated etcd dependencies to v3.6.0-alpha.0 and server to a new commit.
  • Bumped google.golang.org/protobuf and related gRPC/genproto dependencies.
  • Updated various other dependencies (cobra, pflag, golang.org/x/*, cel.dev/expr, etc.).
  • Modified LICENSE files for etcd vendored code to reflect correct copyright.

File Changes

  • go.mod — modified
    Updated dependency versions for etcd, protobuf, gRPC, and other libraries.
  • staging/src/k8s.io/code-generator/examples/go.mod — modified
    Replaced etcd dependencies with github.com/fuweid/etcd fork and updated other dependencies.
  • LICENSES/vendor/go.etcd.io/etcd/api/v3/LICENSE — modified
    Updated copyright and license hash for etcd API.
  • staging/src/k8s.io/apiserver/go.mod — modified
    Bumped etcd, protobuf, gRPC, and other dependencies.
  • staging/src/k8s.io/component-base/go.mod — modified
    Updated dependency versions for cobra, pflag, opentelemetry, grpc-gateway, and others.

Risks & Impact

  • Potential incompatibility with the custom etcd fork (github.com/fuweid/etcd).
  • Upgrading protobuf and gRPC may introduce subtle API or wire format changes.
  • Dependency bumps could cause downstream build or runtime issues.
  • License changes must be validated for compliance.
